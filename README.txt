Easy Italian Fiscality for WooCommerce: checkout fields, invoices, digital goods and everything you need for a store based in Italy

== Description ==

The Italian fiscality and also the European too requires specific duties in order to fit well in WooCommerce.

Those duties can be very tricky to implement and can require different plugin with a specific adaption.
This is not the easy ecommerce solution that we wanted!
Codeat created this plugin in order to resolve all the major issues without needing any coding skill at all!

Install it, activate it, fill all the store info's fields and the job is done. Our plugin will make everything easy for you. 

Free version features:

* Checkout and Orders extra fields (VAT/SSN)
* Italian Tax Rate 22%
* Store Info Data
* Terms & Conditions page check
* ODR (Online Dispute Resolution)
* Shipping and Payment Methods Pages
* Free shipping for customers in the same ZIP code of the shop
* WPML support

Pro version:

* Extra fields in invoice generated by WooCommerce PDF Invoices & Packing Slips (also PRO) or YITH WooCommerce PDF Invoice and Shipping List (also Premium)
* Sequential invoice number
* Company Data (Shortcodes and other stuff for better handling informations on front-end)
* PDF Terms & Conditions as email attachment
* Cash on delivery additional cost
* Shipping to islands additional cost
* VAT and VIES validation for your customers
* Export orders (single or as a zip)

Available Addon:

* Digital Goods EU VAT Rates (in order to selling digital products across Europe)

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'woo fiscalita italiana'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `woo-fiscalita-italiana.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `woo-fiscalita-italiana.zip`
2. Extract the `woo-fiscalita-italiana` directory to your computer
3. Upload the `woo-fiscalita-italiana` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
